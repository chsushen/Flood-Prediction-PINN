# Flood Prediction using Physics-Informed Neural Networks (PINNs)

## Overview
This project focuses on flood prediction using Physics-Informed Neural Networks (PINNs), combining deep learning with physical laws.

## Objective
To improve prediction accuracy by integrating domain-specific physical constraints into neural networks.

## Methodology
- Neural Networks (TensorFlow/Keras)
- Physics-based loss functions
- Hydrological data modeling

## Features
- Combines data-driven and physics-based approaches
- Handles complex environmental patterns
- Suitable for data-scarce regions

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib

## Status
Ongoing project. Research paper submitted to IEEE RMKMATE 2026 (under review).
